# Evaluarea3ToderasAndreiNicolae
Viziunea Produsului (Google Drive)


Oportunitatea și problema: Utilizatorii au nevoie de acces la documente de pe orice dispozitiv, fără a depinde de stocare fizică.


Poziționarea produsului: O bibliotecă de resurse digitale securizată, bazată pe cloud.


Stakeholders: Google Cloud Team, Administratori IT.


Utilizatori: Studenți, Profesori, Editori de conținut.


Capabilități: Sincronizare automată, editare colaborativă și căutare inteligentă.

2. Specificații Software 


Specificație Funcțională: Sistemul trebuie să permită setarea a 3 niveluri de acces: Viewer, Commenter și Editor.

Livrabil Backlog: https://github.com/users/ToderasAndreiNicolae/projects/2


Specificație de Sistem: Aplicația trebuie să asigure criptarea datelor în repaus și în tranzit.


Specificație de Interfață: Interfața web trebuie să suporte funcția de Drag-and-Drop pentru încărcarea fișierelor.
